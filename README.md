# Posturer
Posture diagnostics, analysis and improvement.

## Stack

* **API:** `ASP.NET Web API 2`

* **Web:** `Angular 5`

* **Android:** `Xamarin Android` 

* **IoT:** `Arduino`

* **IoT Emulator:** `JQuery`
